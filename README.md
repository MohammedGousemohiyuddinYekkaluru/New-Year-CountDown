
# New-Year-CountDown

Simple, clean countdown to the next year — HTML, CSS and vanilla JavaScript. Minimal, responsive, and easy to drop into any static site.

## Usage

- Clone or download the repo:

- git clone https://github.com/MohammedGousemohiyuddinYekkaluru/New-Year-CountDown.git


- Open index.html in your browser or use a local server

--- 
## Files

- index.html — markup and DOM placeholders for the timer

- style.css — styling and responsive layout

- script.js — countdown logic and DOM updates
